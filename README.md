Open Terminal and run this command to install PHP `winget install PHP.PHP.8.4`

Download and install composer from here https://getcomposer.org/Composer-Setup.exe

Run this command in the terminal to add stripe dependency `composer require stripe/stripe-php`

Run this command in the terminal to start the PHP server `php -S localhost:8000`

Then go to `localhost:8000/index.html` to go live
